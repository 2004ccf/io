惠州利元亨 · 电气装配岗

一、岗位核心定位

主要负责利元亨新能源（锂电池专用）非标自动化设备的电气系统装配、现场布线、气路安装，以及通电前检查和基础异常排查。说白了，就是把电气元件、线缆、气管，按图纸和车间工艺要求，一步步装到设备上，确保电气系统能正常工作，衔接机械装配和整机调试，不耽误设备出厂。

二、具体实践内容（一线实操全流程，无空泛表述）

（一）前期准备与物料核对（实操第一步，错料就返工）

  1. 图纸熟悉：先领对应设备的电气原理图、接线图、端子图和 BOM 表，必须核对图纸版本和设备型号一致，不然装错了全白费。重点看清楚强电（380V/220V）和弱电（24V）的接线区域，哪个端子接什么线、哪个元件装在什么位置，都要在图纸上标出来，避免后续混乱。
  2. 物料领取与核对：拿着 BOM 表去物料区领料，每一样都要核对规格、型号、数量，一点都不能错。比如断路器，要分清是 16A 还是 20A；线圈电压是 220V 还是 24V；线缆要按线径分好，1.0mm² 的多用于信号线，2.5mm² 的用于动力线，错一根线，后面通电就可能烧元件。还有端子、扎带、线号管这些辅料，也要核对清楚，少一样都没法开工。
  3. 工具检查：万用表要先试一下，通断档、电压档都要正常，避免测的时候出误判；剥线钳要调节好档位，不然剥线要么伤线芯，要么剥太短接不上端子；压线钳、螺丝刀、线号机也要检查，金属工具要注意绝缘，防止触电。

（二）电气柜装配与接线（核心实操，最费时间也最关键）

  1. 元件固定：先在电柜里装导轨，导轨要固定牢固，不然后期元件会松动。然后把断路器、接触器、继电器、PLC、伺服驱动器这些元件卡到导轨上，拧紧固定螺丝的时候，要对角分步拧，不能一次性拧死一颗，不然导轨会变形，元件也会歪。元件之间要留 5–10mm 的空隙，方便散热，也方便后期检修，别挤太满。
  2. 线号打印与线缆处理：按接线图用线号机打印线号管，比如 “PLC-COM1”“KM1-13”，线号要清晰，套在线缆两端，长度要一致，不能歪歪扭扭。剥线的时候，根据线径调剥线钳，剥 3–5mm 就行，剥太长会露线芯，容易短路，剥太短端子夹不住，会虚接。剥完线芯要捋直，不能有毛刺，不然压接的时候会接触不良。
  3. 端子压接与接线：把剥好的线缆压上端子，压的时候力度要刚好，压完用手扯一下，确保不会掉，虚接是电气装配的大隐患，后期设备运行时会发热、打火，甚至烧端子。接线的时候，强电和弱电要分开走，强电走电柜左侧，弱电走右侧，不能混捆在一起，不然强电会干扰弱电信号，设备会误动作。线束要用扎带捆好，每 20cm 捆一道，拐角处要固定好，别让线束拉扯。接地端子要单独接线，必须用黄绿双色线，不能和零线混接，接地电阻要符合要求，不然会有触电风险。
  4. 柜内自检：接完线后，一定要逐根核对线号，确保和图纸一致，不能错号、漏号。然后用万用表测通断，确认没有短路，再检查一遍所有螺丝，尤其是端子和元件的固定螺丝，防止松动。最后把柜内线束整理整齐，贴上元件标识，让人一眼就知道每个元件是什么、干什么用的。

（三）现场设备布线与气路安装（跟着设备走，灵活调整）

  1. 现场布线规划：根据设备机架的结构，还有机械模组、机械手的位置，规划线缆走向，不能挡着运动部件，不然设备运行的时候会磨破线缆。信号线、动力线、传感器线要分开走，不能捆在一起，比如光电传感器的信号线，要是和动力线混捆，信号会受干扰，检测就不准了。线缆要用线管保护，线管固定牢固，拐角处用弯头，别硬折，不然会把线缆弄断。
  2. 现场接线：把电柜里的线缆引出来，穿过线管、线槽，接到现场的伺服电机、传感器、电磁阀上。接线的时候，端子要拧紧，线缆要留 10–15cm 的余量，方便后期调试和维护，别拉太紧，不然设备运行时拉扯线缆，会导致接线松动。传感器接线要注意正负极，比如接近传感器正负接反，检测不到信号，还可能把传感器烧坏。
  3. 气路安装：领取气管和接头，按气路图裁切气管，切口必须垂直，不能斜，也不能有毛刺，斜口会导致接头密封不严，肯定漏气。把气管插入接头，拧紧螺母，力度别太大，不然会把密封圈拧坏，反而漏气。气管要按气路走向固定，用扎带捆好，避开运动部件和高温区域，比如靠近伺服电机的地方，要留足距离，防止气管被烫坏。
  4. 气路试漏：通气的时候，压力调到 0.4–0.6MPa，然后用肥皂水涂抹所有接头、电磁阀接口，要是有气泡冒出来，就说明漏气，要重新拧紧接头或者更换密封圈。试漏合格后，测试气缸、夹爪的动作，确保动作顺畅，没有卡顿、异响。动作不顺通常是气管接错了，或者电磁阀有问题。

（四）通电检查与基础调试（安全第一，逐步推进）

  1. 通电前准备：通电前一定要再检查一遍，确认电柜内接线正确，没有短路、虚接，接地和绝缘都没问题，用万用表测绝缘电阻，至少要大于 1MΩ。还要确认电源电压和设备要求一致，380V 三相电不能接错相序，不然伺服电机转向会反。先断开所有负载，只接通控制电源，避免负载短路烧元件。
  2. 控制电源调试：接通控制电源后，看 PLC、伺服驱动器、指示灯有没有正常通电，PLC 不能有报警提示，指示灯显示要正常。然后测试按钮开关和急停开关，按按钮的时候，对应的指示灯要亮，急停开关按下后，设备要立即断电，确保急停有效。
  3. 负载通电调试：逐步接通负载，先点动测试伺服电机，看转向对不对，要是转向反了，就调换电源相序。然后测试气缸动作，看是不是和控制信号匹配，光电传感器检测的时候要灵敏，能准确检测到工件，压力传感器的数值要准确。遇到简单的问题，比如传感器没信号，先检查接线有没有松动，线号对不对，要是解决不了，就及时找师傅，别自己瞎拆。

（五）现场管理与异常记录（日常必做，不能马虎）

  1. 物料管理：剩余的物料要及时退库，做好记录，不能随便丢在工位上。常用的物料要分类摆放，贴好标识，比如端子按规格分开放，线缆按线径整理好，做到账物相符，不浪费辅料，比如扎带、线号管，别用一半就扔。
  2. 现场 7S 管理：工位要保持干净，工具用完后要归位，螺丝刀、剥线钳要放在指定的盒子里，不能乱丢，金属工具要是掉进电柜，会导致短路。线缆、辅料要摆放整齐，地面不能有油污、杂物，电柜和设备表面要擦拭干净，贴上设备标识和操作规范。施工产生的废料，比如线皮、扎带边角，要及时清理，不能堆积。
  3. 异常记录：实操的时候，遇到元件损坏、接线错误、气路漏气这些问题，要及时记录下来，写清楚问题现象、发生位置，比如 “3 号设备伺服电机接线虚接，导致电机不转”，然后反馈给师傅或班组长，不能擅自处理复杂问题。每天要记录自己做了什么、进度怎么样、遇到了什么问题、怎么解决的，形成工作台账，方便后续查阅。

三、实操关键点（一线经验总结，避坑必看）

（一）电气接线关键点（错一步就返工，甚至出隐患）

  1. 线号是底线：必须和图纸完全一致，错一根线，通电就可能烧 PLC、烧伺服，返工起来特别费时间。接完线一定要逐根核对。
  2. 端子压接必须牢固：压完一定要扯一下，确认不松动，虚接是最大的隐患，后期设备运行时会发热、打火，甚至引发安全事故。
  3. 强弱电必须分开：间距至少 10cm，不能混捆，不然强电干扰弱电，设备会误动作，比如气缸乱动作、传感器检测失灵。
  4. 剥线长度要控制好：3–5mm 最合适，不能伤线芯，也不能剥太短，线芯外露会短路，剥太短接不上端子。
  5. 接地线必须用黄绿双色线：单独接接地端子，不能和零线、信号线混用，接地电阻要小于 4Ω，不然会有触电风险。

（二）气路安装关键点（漏气是常犯问题，重点把控）

  1. 气管切口必须垂直、无毛刺：斜口肯定漏气，裁切的时候要慢一点，切完用手摸一下，没有毛刺再用。
  2. 接头拧紧力度要适中：太松漏气，太紧会损坏密封圈，通气后一定要用肥皂水试漏，没有气泡才算合格，别嫌麻烦。
  3. 气管走向要避开运动部件：比如机械手、升降平台，不然设备运行时会拉扯、磨损气管，时间长了气管会破裂漏气。
  4. 电磁阀接线要正确：正负极不能接反，不然气动部件不动作，甚至会烧坏电磁阀。

（三）安全操作关键点（保命底线，绝对不能违规）

  1. 通电前必须检查绝缘、接地，确认无短路、虚接，严禁带电接线、拆线。带电作业必须有师傅在旁边监护，自己不能单独操作。
  2. 操作的时候必须戴绝缘手套、护目镜，剥线的时候会有铜丝飞溅，接线的时候要防止触电，尤其是接强电的时候，一定要小心。
  3. 万用表使用前要确认档位，别用电压档测通断，不然会损坏万用表。测电压的时候，要先选高档位，再逐步调低。
  4. 急停开关必须保持灵敏，实操的时候，只要遇到紧急情况，不管什么问题，先按下急停，再排查问题。
  5. 金属工具不能掉进电柜：螺丝刀、扳手要是掉进电柜，会导致短路，烧元件。工具用完一定要归位。

（四）工艺规范关键点（利元亨车间要求，必须遵守）

  1. 所有操作都要按利元亨的工艺规范来，不能自己想怎么装就怎么装：比如线束捆扎间距、元件安装位置，都有明确要求，擅自更改会影响设备质量。
  2. 线束要捆扎整齐、美观，间距均匀，拐角处不能折太死，线缆不能扭曲、拉扯，不然后期调试和维护会很麻烦。
  3. 元件安装要牢固，螺丝要拧紧，不能松动、歪斜，要预留足够的检修空间，不然后期检修的时候连螺丝刀都伸不进去。
  4. 所有标识都要清晰：元件标识、线号、设备标识，都要贴整齐，让人一眼就能看明白，方便后期维护和调试。

四、岗位必备能力与知识储备（一线岗位真实要求，不夸大）

  1. 识图能力：能看懂电气原理图、接线图、端子图和 BOM 表，不用师傅手把手教，能根据图纸找到元件位置、规划接线和布线。
  2. 动手实操能力：能熟练用万用表、剥线钳、压线钳这些常用工具，能独立完成电柜装配、接线、现场布线、气路安装和通电前检查，遇到简单问题能自己排查。
  3. 安全意识：懂用电安全、气路安全，能识别电气、气路隐患，不违规操作，知道什么能做、什么不能做，避免出安全事故。
  4. 细节把控能力：做事细心，接线、压接、固定这些细节都能做到位，不犯错线、虚接、漏气这些低级错误，确保实操质量。
  5. 沟通配合能力：能和机械装配、整机调试的同事配合，比如机械装完机架，要及时对接布线，遇到问题能及时反馈，不耽误进度。
  6. 知识储备：懂一点电工基础，知道电压、电流、电阻、绝缘的基本概念，认识常用的电气元件，了解利元亨锂电池非标设备的基本结构。
