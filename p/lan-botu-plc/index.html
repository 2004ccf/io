基本指令

常规指令 ![](attachments/Pasted image 20260115100302.png)

序号名称功能说明
1插入程序段在梯形图中新建一个空白的程序段（Network），用于组织代码
2空功能框插入一个空的功能块框架，可以填入任意指令或功能块
3打开分支创建并联支路的起点，让多个条件可以"或"的关系连接
4嵌套闭合关闭并联支路，把分支合并回主线
5连接线 ─┤画一条横线，用于连接触点、线圈或延长电路

常规指令例子：

  • 插入程序段：每个独立的控制逻辑放一个程序段，比如"启停控制"放Network1，“报警"放Network2
  • 空功能框：先占个位置，后面再填具体指令，比如定时器、计数器
  • 打开分支：启动按钮OR自保持，两个条件任一满足都能输出
  • 嵌套闭合：分支结束后回到主线继续串联其他条件
  • 连接线：把两个元件连起来，或者延长线路到下一个触点

打开分支和嵌套闭合详解：

1
2
3

        ┌─ I0.0 ─┐        打开分支：从这里开始分叉
母线 ───┤        ├─── Q0.0
        └─ I0.1 ─┘        嵌套闭合：在这里合并

意思是：I0.0 或 I0.1 任一个=1，Q0.0就输出 这就是"或"逻辑（OR），用于：

  • 多个启动按钮（主站启动OR副站启动）
  • 自保持电路（启动按钮OR输出自己）

触点类指令

序号名称符号功能说明
1常开触点─┤ ├─信号=1时通，信号=0时断。最常用的触点，用于检测某个信号是否为ON
2常闭触点─┤/├─信号=0时通，信号=1时断。与常开相反，常用于检测信号是否为OFF
3取反─┤NOT├─把前面的逻辑结果反过来（1变0，0变1）

触点类例子：

  • 常开触点：按下启动按钮(I0.0=1)，电机(Q0.0)转。 I0.0 ─┤ ├─ Q0.0
  • 常闭触点：急停没按下(I0.1=0)才允许运行。 I0.1 ─┤/├─ Q0.0
  • 取反：传感器没检测到(I0.2=0)时报警。 I0.2 ─┤NOT├─ Q0.1

线圈类指令

序号名称符号功能说明
4普通线圈─( )─条件满足就ON，不满足就OFF。最基本的输出方式
5取反线圈─(/)─条件满足就OFF，不满足就ON。与普通线圈相反
6复位线圈─(R)─条件满足时，把输出强制变成0。用于清除置位状态
7置位线圈─(S)─条件满足时，把输出强制变成1，之后一直保持。断电或复位才会清除

线圈类例子：

  • 普通线圈：按住按钮灯亮，松开灯灭。 I0.0 ─┤ ├─( Q0.0 )
  • 取反线圈：按钮按下时灯灭，松开时灯亮。 I0.0 ─┤ ├─(/ Q0.0 )
  • 置位线圈：按启动按钮，电机一直转。 I0.0 ─┤ ├─(S Q0.0) 松手也不停
  • 复位线圈：按停止按钮，电机停。 I0.1 ─┤ ├─(R Q0.0) 清除置位状态

置位/复位功能块

序号名称功能说明
8SET_BF置位位域，一次性把多个连续的位都设为1
9RESET_BF复位位域，一次性把多个连续的位都设为0
10SR置位优先触发器。当S和R同时为1时，输出=1（置位优先）
11RS复位优先触发器。当S和R同时为1时，输出=0（复位优先）

置位/复位例子：

  • SET_BF：一键把Q0.0到Q0.7全部置1（8个灯全亮）
  • RESET_BF：一键把Q0.0到Q0.7全部清0（8个灯全灭）
  • SR：启动和停止同时按下，电机转（启动优先）
  • RS：启动和停止同时按下，电机停（停止优先，更安全）

  6. ─┤P├─ 上升沿 - 检测信号变化的瞬间 → 例：计数、切换

定时器操作 ![](attachments/Pasted image 20260115102558.png) IEC标准定时器（常用）

序号名称功能说明
1TP脉冲定时器：输入ON后，输出固定时长的脉冲，时间到自动OFF
2TON接通延时：输入ON后，延时一段时间才输出ON
3TOF关断延时：输入OFF后，延时一段时间才输出OFF
4TONR时间累加器：可暂停的定时器，输入OFF时暂停计时，ON时继续

IEC定时器例子：

  • TP：按一下按钮，灯亮5秒后自动灭（固定脉冲）
  • TON：按下启动3秒后电机才转（防误触）
  • TOF：松开按钮后风扇再转10秒才停（延时关闭）
  • TONR：累计运行时间，中途停机不清零

小白必须掌握的3个定时器：

  1. TON 接通延时 - 最常用，按下后延时动作 → 例：启动延时、防抖动
  2. TOF 关断延时 - 松开后延时停止 → 例：风扇延时关闭、润滑延时

定时器使用要点：

  • ET：已计时时间（Elapsed Time）
  • Q：定时器输出
  • IN：定时器输入/启动条件

计数器操作 ![](attachments/Pasted image 20260115102541.png) IEC标准计数器（常用）

序号名称功能说明
1CTU加计数器：每来一个脉冲，计数值+1，达到设定值时输出ON
2CTD减计数器：每来一个脉冲，计数值-1，减到0时输出ON
3CTUD加减计数器：可以加也可以减，有两个输入分别控制加和减

IEC计数器例子：

  • CTU：数产品数量，每过一个产品+1，数到10个输出信号
  • CTD：倒计时，从10开始减，减到0报警
  • CTUD：入库+1出库-1，实时显示库存数量

线圈型例子：

  • QD：减到0时输出ON

小白必须掌握的10个运算指令：

  1. CALCULATE 计算 - 万能计算器 → 例：复杂公式一次搞定
  2. ADD 加法 - 最基本的运算 → 例：温度补偿、数量累加
  3. SUB 减法 - 计算差值 → 例：剩余量、偏差计算
  4. MUL 乘法 - 比例计算 → 例：单价×数量、比例换算
  5. DIV 除法 - 求平均值 → 例：平均温度、速度计算

  10. LIMIT 限幅 - 防止超限 → 例：输出限制在0~100%

运算指令使用要点：

  • 数据类型要匹配：整数用INT，小数用REAL
  • 除法要防止除零错误：先判断分母≠0
  • 浮点数比较要用范围：不要用==，用ABS(A-B)<0.01
  • 溢出保护：结果可能超出变量范围要检查
  • CALCULATE最灵活：可以写复杂表达式，但要注意运算优先级
  • INC/DEC最简单：不需要输入参数，直接对变量操作

  • SHL：乘法运算 数值左移1位 = 乘以2 快速乘法
  • ROR：数据轮换 循环右移实现数据轮换显示
  • ROL：循环移位 循环左移实现流水灯效果

原有指令例子：

  • DRUM：步骤控制 按时间或条件顺序执行各个步骤
  • SEG：数码管 把数字0~9转换成7段数码管显示码
  • BITSUM：位计数 统计报警字中有多少个报警

小白必须掌握的4个移位指令：

  1. SHL 左移 - 快速乘2 → 例：数值×2，流水灯左移
  2. SHR 右移 - 快速除2 → 例：数值÷2，数据压缩
  3. ROL 循环左移 - 数据轮换 → 例：流水灯循环，数据轮显
  4. BITSUM 位计数 - 统计1的个数 → 例：统计报警数量

移位指令使用要点：

  • 移位是快速乘除法：左移1位=×2，右移1位=÷2
  • 循环移位保持数据：数据不会丢失，只是位置变化
  • 流水灯用循环移位：ROL实现灯光循环效果
  • 位统计用BITSUM：快速统计状态位数量
  • 注意移位位数：移位位数不能超过数据宽度
